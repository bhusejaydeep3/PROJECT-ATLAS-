# Project Atlas 

## Vision 

Project Atlas is an autonomus AI operating system desined to plan , reason,execute,learn 

Atlas is built with the following principles:

-Local-first 
-Free and open-source whenever possible 
-Modular architecture 
-Explainable decision making 
-Human oversight for critical actions 

Atlas is not a chatbot.
Atlas is an intelligence operating system capable of achieving long-term goals through autonomous planning and execution.
